# Könyvtár Kezelő Rendszer

Egy teljeskörű, adatbázis-alapú könyvtári nyilvántartó rendszer **PHP + MySQL + CSS**-sel.  
Könyvek hozzáadása, keresése, listázása — reszponzív, hover effektusok, árnyékok.

## Funkciók
- Könyv hozzáadása (cím, szerző, kiadás éve, műfaj)
- Keresés cím vagy szerző alapján
- Teljes könyvek listája táblázatban
- Modern, hoverolható gombok + árnyékok (CSS)
- MySQL adatbázis integráció (XAMPP)

## Technológia
- **PHP** – backend logika
- **MySQL** – adatbázis (XAMPP)
- **HTML/CSS** – reszponzív dizájn, hover effektek
- **6 PHP fájl + 1 CSS**

## Telepítés (XAMPP)

1. Indítsd el az **Apache**-et és **MySQL**-t XAMPP-ban
2. Hozz létre adatbázist: `konyvtar`
3. Importáld a `database.sql` fájlt (ha van)
4. Másold a fájlokat: `htdocs/konyvtar`
5. Nyisd meg: `http://localhost/konyvtar`
